## 🥁 [선택 1] 드럼 스튜디오 예약 & 쇼룸 랜딩 페이지 프롬프트

### 📋 복사용 프롬프트 (Prompt)

```text
Please build a premium, modern single-page web application for a high-end drum studio named "BEAT & VINYL" (드럼 스튜디오 & LP 라운지).

[Design & Aesthetics]
- Color Palette: Sleek dark mode. Deep charcoal/black background (#121212), warm amber/neon yellow (#FACC15) as the accent color, and soft gray for text.
- Typography: Use a modern sans-serif font (like Inter or Outfit) with bold headings and spacious layouts.
- Vibe: Premium, cozy, and music-oriented (combining raw acoustic drum vibe with warm LP analog aesthetics).

[Sections]
1. Header/Navigation: Logo "BEAT & VINYL", and simple scroll links (소개, 대여 계산기, LP 쇼룸, 문의하기).
2. Hero Section:
   - Big bold headline: "소음 민원 제로. 50cm 더블 방음벽을 갖춘 프리미엄 드럼 스튜디오"
   - Subtitle: "압도적인 방음 부스와 빈티지 LP 청음 라운지가 결합된 음악인들의 아지트"
   - Call to Action button: "실시간 대여 계산기 바로가기" (smooth scroll to Section 4).
3. Space Showcase: 
   - 3 visually stunning cards for:
     * "드럼 작업실 (녹음/합주/레슨)": 전문 녹음 및 개인 연습용 최상급 세팅 (외경 6m x 5m)
     * "월 임대 연습실": 전공생들을 위한 24시간 지정 대여 연습실
     * "LP 청음 & 감성 라운지": 따뜻한 조명 아래 턴테이블과 음악 잡지가 있는 휴식 공간
4. Interactive Price Calculator (대여 요금 계산기):
   - Users can calculate booking rates dynamically:
     - Select Room Type: 드럼 작업실 (시간당 15,000원) / 개인 연습실 (시간당 10,000원)
     - Select Booking Hours (Slider from 1 to 10 hours)
     - Checkbox options (Add-ons): 
       * 드럼 스틱 대여 (+2,000원)
       * 음료 이용권 (+3,000원)
     - Display the dynamically calculated "Total Price" (총 예상 금액) with a smooth counter effect.
     - Add a "예약 신청하기" button that triggers a success modal.
5. LP Curation Shop (이달의 LP 쇼룸):
   - Grid layout showing curated vinyl albums with album arts, artist names, and descriptions:
     * Beatles - Abbey Road
     * Unknown Mortal Orchestra - II
     * Supercar - Futurama
   - Each card has a hover effect and a "청음 해보기" play/pause mock button.
6. Footer: Location (예: 인천 부평구), business register details, and social links.

[Technical Requirements]
- Use clean HTML, CSS, and modern Javascript.
- Make the layout fully responsive for mobile and desktop devices.
- Include smooth scroll transitions and hover animations for all buttons and cards.
```

## 🗺️ [선택 3] '세상의 모든 지하철' 도심 도보 & ASMR Curation 웹맵 프롬프트

### 📋 복사용 프롬프트 (Prompt)

```text
Please build a modern, interactive single-page web application for a city walking tour and background ASMR platform named "세상의 모든 지하철" (Subway ASMR & City Walk Curation).

[Design & Aesthetics]
- Color Palette: Clean dark mode. Pure black (#000000) and deep gray (#1F2937) background. Accent colors should match Seoul Subway lines (Line 1: #003499, Line 2: #37b42c, Line 6: #cd7c2f).
- Vibe: Minimalist, urban, and calm. Think of a high-end audio curation site or premium travel archive.

[Sections]
1. Header: Logo "세상의 모든 지하철" and a slogan "가장 현실적인 도시의 10분, 역 주변의 공기를 담다."
2. Subway Line Tabs (노선 필터):
   - Clean tab buttons for: "전체", "1호선", "2호선", "6호선".
   - Clicking a tab filters the station cards below with a smooth fade-in animation.
3. Station ASMR Curation Grid (역별 아카이브):
   - Display cards for featured subway stations:
     * [Line 2] 강남역 11번 출구 (금요일 저녁 7시의 활기참)
     * [Line 2] 망원역 2번 출구 (망원시장으로 향하는 망원동 골목길)
     * [Line 6] 상수역 1번 출구 (상수동 카페거리의 잔잔한 저녁)
     * [Line 1] 종로3가역 3번 출구 (포장마차 거리의 정겨운 대화 소리)
   - Each card must contain:
     - A beautiful placeholder thumbnail image representing the neighborhood.
     - A mini audio player control: "Play/Pause" button, a animated audio waveform visualizer (using CSS animations to show playing state), and a mock time progress bar.
     - A "도보 코스 보기" badge that expands/toggles a text description of the recommended 10-minute walk route.
4. Interactive Walking Map Widget:
   - A visual section showing a stylized mock map (or SVG path) of a selected walking route.
   - Show stats: "총 거리: 850m", "소요 시간: 10분", "주요 화이트노이즈: 발걸음 소리, 자전거 경적, 골목길 낙엽 소리".
5. Footer: YouTube Channel subscription CTA link and social links.

[Technical Requirements]
- Utilize responsive grid layouts.
- Interactive Javascript for:
  - Filtering cards when subway line tabs are clicked.
  - Toggling play/pause state for the audio player on each card (triggering CSS waveform animation when playing).
  - Toggling the walking route details panel.
- Ensure smooth hover scale effects for station cards.
```
